# agent-skills

A public, curated library of skills for AI agents.

Each skill lives in its own subdirectory under this repository and follows the Agent Skills format.

## Install a skill

```bash
npx skills@latest add LegionForge/agent-skills/lf-hermes-memory-addon
```

Or install globally to share across all your projects:

```bash
npx skills@latest add LegionForge/agent-skills/lf-hermes-memory-addon --global
```
